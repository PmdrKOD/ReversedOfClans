Reversed OF clans

## 📋 Системные требования

Для корректной работы приложения необходимо наличие следующего ПО:

- **.NET 9.0** (Обязательно)
- ОС: Windows / Android (с ограничениями, см. ниже)

## ⬇️ Загрузка клиента
👉  Скачать клиент здесь ->  https://drive.google.com/file/d/1jk0tXvqMuWGOpkpVGUhq5jVWZxNAtqft/view?usp=drive_link>

## ⚠️ Важная информация о запуске

Обратите внимание на следующие ограничения при запуске приложения:

1. **Совместимость с Android:** Клиент поддерживает работу только на **старых версиях Android**. На новых устройствах запуск может быть невозможен.
2. **Рекомендация:** Для стабильной работы настоятельно рекомендуется использовать эмулятор Android.
   - ✅ **Рекомендуемый эмулятор:** [LDPlayer](https://www.ldplayer.net/)
   - Настройте эмулятор на соответствующую версию Android (например, Android 5.1 или 7.1)

## 🚀 Как запустить

1. Установите **.NET 9.0 Runtime**, если он еще не установлен.
2. Скачайте сурс с клиентом по ссылке выше.
3. Распакуйте файлы в отдельную папку.
4. Смените айпи адрес в клиенте в файле libs.so
5. Запустите исполняемый файл.
   - *Если вы на ПК:* Запускайте через эмулятор LDPlayer (см. раздел выше).
   - *Если вы на устройстве:* Убедитесь, что версия Android совместима.
